DRAW 2 GETHER

Draw 2 Gether is an online multiplayer drawing game. In the game, a user is given a quadtrant of an image to draw (ex. top left), as well as an item to draw (ex. house). The user then has access to a canvas to draw on, in addition to an RGB slider and an eraser to make the picture their own. Once finished, the user can save their photo, and then view it as a completed photo along with 3 other photos stitched together. There is also a gallery of many different photos that users have created. The game was made using Java and XML in the Android Studio environment, as well as Microsoft Azure to store and stitch together the pictures.
